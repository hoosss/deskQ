# deskQ 🐾

> An AI desktop companion built with Rust + Tauri.

deskQ is an open-source AI desktop pet. It lives on your desktop as an animated
companion and can integrate with LLMs (local or remote) for conversation,
reminders, and ambient assistance — privacy-first.

## Status

🚧 Early scaffold — work in progress.

## Tech Stack

| Layer | Choice |
|-------|--------|
| Shell / backend | Rust |
| UI / sprite animation | Tauri 2 + Web frontend |
| AI | Pluggable LLM provider (local-first) |
